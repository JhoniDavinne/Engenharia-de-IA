# Arquitetura RAG com LLMs

## Introdução e objetivos

Este curso é destinado a pessoas que desejam resolver problemas e superar a **falta de contexto dos LLMs**, utilizando a técnica de **RAG** (*Retrieval Augmented Generation* — Geração Aumentada por Recuperação), para ampliar o contexto e melhorar as respostas dos LLMs com **contextos externos**.

## O que você vai ver no curso

- **Vector Store** (armazenamento vetorial)
- **Embeddings** (incorporação de dados)
- **Chunks** (fragmentos de texto)
- **Metadados** — como incluí-los no fluxo

Tudo isso **orquestrado com LangChain**.

## Projetos práticos

- Chat inteligente sobre **regras de futebol**
- Chat inteligente sobre **bulas de remédios** em uma farmácia
- Chat para **dúvidas corporativas**, apoiando o **RH** no atendimento aos colaboradores

## Participação

Durante o curso, orquestramos **RAG** com **LangChain**. Se esta introdução fez sentido para você, o curso aprofunda **RAG com documentos**, usando melhor os dados para enriquecer as respostas dos LLMs.
